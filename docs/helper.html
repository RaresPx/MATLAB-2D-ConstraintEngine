Editor Help Documentation

Introduction

Welcome to the interactive editor! This application allows you to manipulate objects and manage various settings in your scene. Use the toolbar and mode buttons to navigate through different tools and functionalities.

Mouse Interactions

Your mouse is key to interacting with the scene. Below are some of the mouse functions you can use:

  • Left-click: Select or place objects in the scene. Depending on the mode, clicking will either add new objects or select existing ones for manipulation.
  • Right-click: Open a properties panel for the selected object. You can adjust various settings for the object here.
  • Drag: While in the "drag" mode, you can click and hold an object to drag it around the scene.
  • Add Constraint: While in the "addConstraint" mode, left-click to pick a second body and define a constraint between the first and the second body.

To exit any interaction, you can simply release the mouse button, and the object or mode will be deselected. You can then proceed with other actions.

Editor Modes

Each mode provides different functionality. You can switch between modes using the toolbar buttons or hotkeys:

  • Select Mode: Use this mode to interact with objects, select them, and move them around.
  • Add Circle: Adds a circle to the scene when you click in the desired location.
  • Add Rectangle: Adds a rectangle to the scene. Click to place the rectangle.
  • Add Constraint: Allows you to create constraints between two objects in the scene. This is used for physics simulations.
  • Delete: Select objects to delete them from the scene.

Keyboard Shortcuts

Here are some useful keyboard shortcuts to make your editing faster:

  • Esc: Closes the application immediately.
  • Spacebar: Toggles the pause state of the simulation.
  • C: Switches to "Add Circle" mode.
  • R: Switches to "Add Rectangle" mode.
  • K: Switches to "Add Constraint" mode.
  • G: Toggles the ground in the scene.
  • D: Switches to "Drag" mode to move objects.
  • Q: Resets the scene to its default state.
  • M: Opens the menu and exits the editor.
